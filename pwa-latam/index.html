🔑 Token GitHub

⚠️ O token fica salvo APENAS neste celular (localStorage). Nunca compartilhe.
Token com permissão de escrita no repositório carlaobh20/cacapassagens + permissão Actions: Read and write
📖 Como criar o token (3 min)
  1. Abra github.com/settings/tokens (Fine-grained tokens)
  2. Clique Generate new token
  3. Nome: cacapassagens-pwa
  4. Expiração: 1 ano ou No expiration
  5. Repository access → Only select repositories → escolha cacapassagens
  6. Permissions → Repository permissions: Contents = Read and write, Actions = Read and write, Metadata = Read
  7. Clique Generate token
  8. Copie o token (ghp_... ou github_pat_...) e cole aqui em cima
